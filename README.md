## Overview
Porte Boshbo is an online platform that connects struggling students with proficient instructors for instant, targeted help with lessons and assignments. It focuses on improving the student learning experience by offering on-demand academic support through virtual one‑to‑one sessions.

## Core Features
- Student–instructor matching based on institution, level, and topic, with private virtual rooms for live sessions.
- 24/7 access to tutoring and assignment help via a subscription model, including options for offline monthly tutoring.
- Instructor profiles with ratings, teaching hours, and student counts, plus post‑session feedback and flexible skip/reshuffle options.

## Tech Stack
- **Backend**: ASP.NET REST API for authentication, matching, sessions, and payments.
- **Database**: MySQL for users, sessions, subscriptions, and ratings.
- **Real‑time & Video**: WebRTC / Socket.io for virtual rooms and live interaction.
